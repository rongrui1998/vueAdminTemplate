# 后续扩展建议

以下内容不属于当前 Base 范围，但可以作为后续 Standard / Pro 的推进方向。

## Standard 方向

- 更完整的页面权限模型
- 用户管理、角色管理、菜单管理
- 字典能力

## Pro 方向

- ProTable
- SearchForm
- ModalForm / DrawerForm 通用封装
- 上传组件封装
- 导入导出能力
- 更完整的全局设置项

## Base 继续优化建议

如果仍继续打磨 Base，可优先考虑：

- Dashboard 增加快捷入口或最近操作区
- 标签页交互增强
- 动态菜单异常配置下的开发态提示
- 增加权限审计或操作日志示例

当前建议是：Base 先保证“能启动、能演示、能复用、能说明白”，再继续进入 Standard 阶段。
